# Simulation Analysis

## 1. Independent Data Simulation Analysis

Run `1.Simulation_independent.R` to calculate FDR, Power, and heterogeneity for PALM and compared methods using simulated independent data.

## 2. Longitudinal Data Simulation Analysis

Run `2.Simulation_longitudinal.R` to calculate FDR, Power, and heterogeneity for PALM and compared methods using simulated longitudinal data.

## 3. Independent Data Simulation Analysis (Report AUPRC)

Run `3.Simulation_AUPRC.R` to calculate AUPRC for PALM and compared methods using simulated independent data.

## 4. Independent Data Simulation Analysis (40% signature proportion)

Run `4.Simulation_effect.R` to calculate calculate FDR, Power, and heterogeneity for PALM and compared methods using simulated independent data by varying effect size.

## 5. Independent Data Simulation Analysis (Report Wald-version PALM)

Run `5.Simulation_PALM_Wald.R` to calculate FDR, Power, and heterogeneity for Wald-version PALM using simulated independent data.
